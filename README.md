# 🎮 Top-Down Multiplayer Prototype

**Unity | Netcode for GameObjects | PC | 2025**

Демонстрационный проект мультиплеерного взаимодействия в top-down перспективе. Основной фокус — реализация плавного сетевого геймплея с минимальными задержками и отзывчивым управлением через современные техники синхронизации.

## 📖 Описание

Прототип, разработанный для проверки ключевых гипотез сетевого взаимодействия и управления игровыми сущностями. В проекте реализованы механизмы, позволяющие нескольким игрокам совместно взаимодействовать с миром, транспортом и NPC в реальном времени.

## 🧩 Ключевые особенности

- **Предсказание и интерполяция (Client-Side Prediction & Interpolation)** — настройка сетевых параметров для плавного движения объектов и снижения влияния латентности на геймплей.
- **Система транспорта** — возможность занимать места водителя и пассажира, синхронизированная между всеми клиентами.
- **Перевозка грузов** — подбор, транспортировка и выгрузка объектов с полной сетевой авторизацией.
- **Динамическое переключение персонажей** — игрок может в реальном времени переключаться между доступными сущностями без прерывания сессии.
- **Синхронизация NPC** — базовая система поведения неигровых персонажей, единая для всех участников сессии.

## 🛠️ Технический стек

- **Движок:** Unity 6 (6000.x)
- **Сетевой фреймворк:** Netcode for GameObjects (NGO)
- **Транспортный уровень:** Unity Transport (UTP)
- **Архитектура:** Client-Authoritative / Server-Authoritative (гибридная модель)
- **Платформа:** Windows (PC)

## 🎮 Управление

| Действие               | Клавиша          |
|------------------------|------------------|
| Передвижение           | WASD             |
| Посадка в транспорт    | E                |
| Переключение персонажа | Tab              |
| Взаимодействие         | F                |


## ⚠️ Известные ограничения

- Проект является прототипом — возможны неоптимальные сетевые вызовы в edge-cases.
- Система NPC синхронизируется, но не содержит продвинутого AI.
- Не реализована полноценная система лобби (используется прямое подключение).
